Homebrew's Python3 framework /usr/local/Cellar/python3/3.3.0/Frameworks/Python.framework

Distribute and Pip have been installed. To update them pip3 install –upgrade distribute pip3 install –upgrade pip

To symlink "Idle 3" and the "Python Launcher 3" to ~/Applications `brew linkapps`

You can install Python packages with `pip3 install <yourfavoritepackage>;`

They will install into the site-package directory /usr/local/lib/python3.3/site-packages Executable python scripts will be put in: /usr/local/share/python3 so you may want to put "/usr/local/share/python3" in your PATH, too.

See: https://github.com/mxcl/homebrew/wiki/Homebrew-and-Python
